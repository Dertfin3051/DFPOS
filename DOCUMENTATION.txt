DF Portable Operation System
Автор : dertfin / ДертФин

Установка :
  | Просто запустите файл dfpos через python3
  | и согласитесь с установкой


Первый запуск:

  | После первого запуска пройдёт небольшая загрузка, после чего вас попросят
  | создать нового пользователя.
  |
  | После создания пользователя вам нужно перезапустить систему и войти в аккаунт

Команды:
  | settings - Настройки:
  | Настройки:
    | exit - Выход
    | change password - Смена пароля
    | change symbol - Смена символа ввода
    | change username - Смена имени пользователя
    | show username - Вкл/выкл отображения имени пользователя
    | help - Помощь
  |
  | system - Система:
  | Система:
    | help - Помощь
    | exit - Выход
    | info - Информация о системе
